# Task 02 - Node.js Backend Server

## 📌 Project Overview

This project is a basic backend application developed using Node.js and Express.js. The primary objective of this project is to understand how backend servers work, how HTTP requests are handled, and how responses are sent from the server to the client.

This project helped me gain practical experience in backend development and server-side programming.

---

## 🎯 Objectives

- Learn the fundamentals of backend development.
- Understand client-server architecture.
- Handle HTTP requests and responses.
- Create and run a web server using Node.js.
- Explore Express.js for server development.

---

## 🚀 Features

- Backend server built using Node.js.
- Handles incoming client requests.
- Sends responses dynamically.
- Demonstrates server-side programming concepts.
- Lightweight and easy-to-understand code structure.

## ⚙️ Installation & Setup

### Step 1: Clone the Repository

```bash
git clone <repository-url>
```

### Step 2: Navigate to the Project Folder

```bash
cd Task02-Backend-Server
```

### Step 3: Install Dependencies

```bash
npm install
```

### Step 4: Run the Server

```bash
node server.js
```

---

## 🌐 Usage

After starting the server, open your browser and visit:

```text
http://localhost:3000
```

The server will process the request and display the response.
